# GrafcetT

Semplice libreria per scrivere codice in Arduino secondo la tecnica Grafcet.

Per l'installazione in Arduino IDE:  
Sketch > Include Library > Add .ZIP Library...  
[GrafcetT.zip](GrafcetT.zip)
